# 📊 Trader Performance vs Market Sentiment Analysis

## 📌 Project Overview

This project analyzes the relationship between cryptocurrency trader performance and Bitcoin market sentiment using historical trading data and the Fear & Greed Index.

The objective is to identify how market sentiment influences trading performance and uncover patterns that can help improve trading strategies.

---

## 📂 Dataset

The project uses two datasets:

1. Historical Trader Data
2. Bitcoin Fear & Greed Index

The datasets were merged using the trading date to associate each trade with the corresponding market sentiment.

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📈 Analysis Performed

- Data Cleaning
- Date Formatting
- Dataset Merging
- Trade Distribution by Market Sentiment
- Average Closed PnL Analysis
- Total Closed PnL Analysis
- Profit vs Loss Analysis
- Top 10 Profitable Coins
- Top 10 Traders
- Buy vs Sell Performance
- Monthly Trading Activity

---

## 📊 Key Insights

- Fear recorded the highest number of trades.
- Extreme Greed produced the highest average profit per trade.
- Fear generated the highest total profit because of higher trading activity.
- More trades ended in loss than profit, but profitable trades generated larger gains.
- HYPE, SOL, ETH, and BTC were among the most profitable cryptocurrencies.
